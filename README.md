# Habitable Planets Explorer

An interactive web application that explores potentially habitable exoplanets and theoretical faster-than-light travel concepts. This application focuses on answering two crucial questions:

1. Which planets beyond our solar system could potentially support human life?
2. How might we reach these distant worlds within a single human lifetime?

## Features

- **Habitable Planets Database**: Information on the most promising potentially habitable exoplanets currently known to science.
- **Interstellar Travel Technologies**: Exploration of theoretical propulsion and travel methods that could enable interstellar journeys within a human lifespan.
- **Journey Calculator**: Interactive tool to calculate travel times to various exoplanets using different propulsion technologies.

## How to Use

1. Simply open the `index.html` file in any modern web browser to launch the application.
2. Explore the different sections using the navigation menu.
3. Use the Journey Calculator to estimate travel times to different planets with various propulsion technologies.

## Scientific Basis

The information presented is based on current scientific understanding, including:

- **Exoplanet Data**: Based on NASA's Exoplanet Archive, ESA's CHEOPS mission, and other sources.
- **Propulsion Technologies**: Based on theoretical physics papers and NASA's Breakthrough Propulsion Physics program.

## Disclaimer

While the exoplanet data is based on actual scientific discoveries, some details are estimated where direct measurements are unavailable. The travel technologies section includes both established propulsion methods and highly theoretical concepts that may not be physically possible.

## Technical Implementation

This application is built with pure HTML, CSS, and JavaScript, making it easy to run locally without any build process or dependencies.
