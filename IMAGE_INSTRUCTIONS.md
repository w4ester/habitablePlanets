# Image Instructions for Habitable Planets Explorer

## Required Images

The website is designed to use the following images. Since I cannot directly generate or download images, you'll need to add them to the project:

### 1. Background Starfield Image

Needed at: `/images/stars-bg.jpg`

A deep space starfield image for the website background. You can find suitable free-to-use images from:
- [NASA Image Library](https://images.nasa.gov/)
- [Unsplash](https://unsplash.com/s/photos/stars-space)
- [Pexels](https://www.pexels.com/search/space%20stars/)

### 2. Planet Images

Needed at:
- `/images/proxima-b.jpg`
- `/images/trappist-1e.jpg`
- `/images/kepler-442b.jpg`
- `/images/toi-700d.jpg`
- `/images/teegarden-b.jpg`
- `/images/k2-18b.jpg`

Artist's conceptions of each exoplanet can be found at:
- [NASA Exoplanet Catalog](https://exoplanets.nasa.gov/discovery/exoplanet-catalog/)
- [European Southern Observatory](https://www.eso.org/public/images/)

## Temporary Solution

Until you add the actual images, the website uses placeholder images with the planet names. The website is fully functional without the actual images, but adding them will enhance the visual appeal significantly.

## How to Add Images

1. Download appropriate images from the sources mentioned above
2. Rename them according to the filenames listed above
3. Place them in the `/images` directory of the project
4. No code changes are required - the website will automatically use these images once they're added

## Image Optimization

For best performance:
- Resize images to appropriate dimensions (planet images around 600×400 pixels, background around 1920×1080 pixels)
- Compress images to reduce file size
- Consider using .webp format for modern browsers if file size is a concern
